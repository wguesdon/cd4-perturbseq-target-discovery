# Hackathon brief and judging criteria

Reference copy of the official rules that shape strategy. Source: the 2026 Built with
Claude: Life Sciences participant guide (Cerebral Valley).

## Track: Researcher, "Build From the Bench"

Using Claude Science, start from a biological question and find the existing datasets and
tools needed to answer it. Submit something discrete: a finding, a trained model, or an
analysis others can reproduce. Show how Claude Science got you there.

Anchor dataset for this project, a Researcher Track example: new drug targets in the CD4+
T cell Perturb-seq data from Alex Marson's lab.

- Dataset: https://virtualcellmodels.cziscience.com/dataset/genome-scale-tcell-perturb-seq
- Code: https://github.com/emdann/GWT_perturbseq_analysis_2025
- Preprint: https://www.biorxiv.org/content/10.64898/2025.12.23.696273v1

## Submission requirements

Due 2026-07-13, 9:00 PM ET, via the CV platform.

1. Demo video, 3 minutes maximum.
2. Open source repository, notebook, or research write up.
3. Written summary, 100 to 200 words.

## Judging criteria

Stage 1 is asynchronous, 2026-07-14 to 07-15. Standardized scoring:

- Impact, 25%. Real world potential. For Researcher projects: is this a finding or
  analysis others can build on? Does it fit the track problem statement?
- Claude Use, 25%. How creatively Claude Code was used. Did it go beyond a basic
  application and surface surprising capabilities?
- Depth and Execution, 20%. Did the team push past the first idea? Sound, refined
  engineering. Real craft, not a quick hack.
- Demo, 30%. A working, compelling demo. Findings you trust. Genuinely cool to watch.

Stage 2 is the final live round, 2026-07-16, 12:00 PM ET. The top teams' pre recorded
demos play, 3 minutes each. Winners are announced at the closing ceremony, 1:30 PM ET.

## What this means for strategy

- Demo is the largest single weight at 30%. Budget real time for a crisp 3 minute video.
  Make the benchmark figure the centerpiece and show the ranked targets and evidence cards.
- Claude Use is 25%. Make the Claude Science leverage visible: literature grounding,
  ground truth curation, and evidence card generation, not just code writing.
- Impact and fit: the track example is literally new drug targets in CD4+ Perturb-seq,
  which the target triage plan matches directly.
- Depth: the pivot from druggability first to a benchmark validated score is concrete
  evidence of pushing past the first idea. Keep that story in the write up.
